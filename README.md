# Gish Abay Sekela Students Association - DBU

This platform is designed for the Gish Abay Sekela Students Association at Debre Berhan University. It connects members, facilitates communication, and streamlines the financial contribution process.

## Project Overview

The main goal of this project is to digitalize the association's activities, making it easier for members to interact and for administrators to manage contributions transparently.

## Key Features

### 1. Community Chat
- A dedicated communication channel where all association members can discuss and share information.

### 2. Contribution & Finance Management
- Members can record their financial contributions.
- **Receipt Upload:** Users can upload screenshots or photos of their payment receipts for verification.
- Transaction history tracking.

### 3. Admin Dashboard
- Secure login for administrators via Email and Password.
- **Approval System:** Admins can review uploaded receipts and approve or reject payments.
- Member management capabilities.

## Tech Stack

This project is built using modern web technologies:
- **Frontend:** React, Vite
- **Styling:** Tailwind CSS, shadcn-ui
- **Language:** TypeScript

## Getting Started

To run this project locally:

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
